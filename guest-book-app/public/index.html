Welcome to the Guestbook App

Connect with others by leaving and reading messages.

How It Works

This app allows you to:

  • Visit Guestbook to see what others have shared.
  • Navigate to New Message to leave a personal note.
  • Use AJAX Message for an instant messaging experience.

Why Use the Guestbook App?

This app is a simple and fun way to connect:

  • Easy to use: Leave a message in just a few clicks.
  • Instant updates: Enjoy real-time updates with AJAX.
  • A shared space to connect and share thoughts.

This project was created as part of a web development course to demonstrate basic CRUD functionality with Node.js, Express, and AJAX.
